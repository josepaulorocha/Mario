Mario World

Constantes

  • VIDA_MAXIMA = 3 (número máximo de vidas dos personagens)
  • TIPOS_DE_POWERUPS = ["Cogumelo", "Flor de fogo", "Estrela", "Pena"] (tipos de power-ups disponíveis)

Classes Principais

Personagem

Propriedades:

  • nome: string
  • vidas: number
  • inventario: string[]

Métodos:

  • printPular() - Exibe mensagem de pulo no console
  • coletarPowerUp(powerUp: string) - Adiciona power-up ao inventário
  • desviar(): boolean - Retorna true com 40% de chance

Inimigo

Propriedades:

  • tipo: string

Métodos:

  • atacar(personagem: Personagem) - Realiza um ataque ao personagem

PowerUps

Propriedades:

  • tipo: string

Métodos:

  • gerarPowerUp(): string - Retorna um power-up aleatório

Arrays:

  • inimigos: Contém 3 instâncias de Inimigo ("Goomba", "Koopa Troopa", "Bowser")
  • personagens: Contém 2 instâncias de Personagem ("Mario", "Luigi")
  • randomArrayInt: Array com 10 números aleatórios entre 1 e 100

Fluxo do jogo

  1. São geradas 10 rodadas (baseadas nos números aleatórios)
  2. Para cada número:
    • Se ≤ 60 (60% chance): Inimigo aleatório ataca Mario
    • Se > 60 (40% chance): Power-up aparece e Mario coleta
  3. O jogo termina se Mario chegar a 0 vidas ou após todas as rodadas
